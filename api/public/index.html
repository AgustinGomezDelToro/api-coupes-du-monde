🏆 Bienvenue sur l'API des Champions du Monde

Cette API fournit des données historiques sur la Coupe du Monde de football : années, champions, meilleurs joueurs et statistiques de matchs.

📚 Endpoints disponibles :

  • /api/years — Liste des années de toutes les Coupes du Monde
  • /api/championsByYear — Champion par édition
  • /api/championMatches — Matchs joués par les champions
  • /api/bestPlayerStats — Statistiques des meilleurs joueurs
  • /api/champions — Pays ayant remporté la Coupe
  • /api/bestPlayers — Joueurs marquants
